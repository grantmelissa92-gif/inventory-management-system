# IT Equipment Inventory Management System

A Python and MySQL-based inventory system for managing IT equipment records.

## Features
- Add equipment
- View equipment
- Search equipment
- Update equipment status
- Delete equipment

## Technologies
- Python
- MySQL
- SQL

## How to Run

1. Install MySQL
2. Run database.sql
3. Install requirements

pip install -r requirements.txt

4. Run the program

python inventory_system.py
